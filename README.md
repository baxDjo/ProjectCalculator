This project implements a basic calculator that supports the following operations, listed by priority:

🟥 High Priority (Core Functions)

Addition — essential arithmetic operation

Subtraction — essential arithmetic operation

Multiplication — essential arithmetic operation

Division (with zero-protection) — essential arithmetic operation

🟧 Medium Priority (Useful Enhancements)

Input validation (prevent invalid numbers, empty input, non-numeric input)

Error handling (e.g., division by zero, format exceptions)

User-friendly console prompts

🟨 Low Priority (Optional Extras)

History of operations

Continuous calculation mode

Unit tests (xUnit / MSTest)

Support for negative numbers and decimals (if not already implemented)

Additional operations such as:

Power (x^y)

Square root
